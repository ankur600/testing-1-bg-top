Issue: Matrix derivatives of determinants fail for scalar expressions

Error: NotImplementedError: noncommutative scalars in MatMul are not supported

Failing Examples
from sympy import MatrixSymbol, Determinant
from sympy.abc import k
X = MatrixSymbol('X', 3, 3)
dX = Determinant(X)

# These cases currently fail:
expr1 = k * dX
expr1.diff(X)  # Raises NotImplementedError

expr2 = 1 / dX
expr2.diff(X)  # Raises NotImplementedError

expr3 = dX**2
expr3.diff(X)  # Raises NotImplementedError
1
Understand the Problem

The issue occurs when trying to differentiate scalar expressions containing matrix determinants with respect to a matrix variable.

Root Cause: The current implementation doesn't handle the case where a determinant appears within scalar operations (multiplication, division, powers).

When SymPy encounters expressions like k * Determinant(X), it tries to use standard scalar differentiation rules, which fail for matrix operations.
